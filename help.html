Approximations of algebraic and transcendental numbers

The following parameters are available :
- approximated number
- opcjonalna name for that number
- max denominator
- language en, or pl
- we mark ✔ better approximations, i.e. at least istotnie times more precise
- approximation of any number given by the user

Example script usage
approx.php?lang=en&limit=1000 - Approximations of all numbers defined in the script
approx.php?name=pi&lang=en&limit=1000000 - Approximations of π for denominators up to one million
approx.php?lang=en&name=pi&istotnie=3 - Approximations of π marking approximations 3 times better
approx.php?any&lang=en - approximations of any number given by user

For the script to work it is necessary to have a PHP server, e.g. XAMPP.
